Japanese

Japanese uses three main writing systems: hiragana, katakana, and kanji. It combines phonetic symbols and characters borrowed from Chinese.

Learn more about Japanese writing on Wikipedia .

Korean

Korean is written using Hangul, an alphabet designed to be easy to learn. Letters are grouped into blocks that represent syllables.

Learn more about Hangul on Wikipedia .

Chinese

Chinese is written using characters that represent meaning rather than individual sounds. Each character usually represents one syllable.

Learn more about Chinese characters on Wikipedia .
